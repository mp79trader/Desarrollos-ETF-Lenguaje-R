# Desarrollos-ETF-Lenguaje-R

## Descripción
Repositorio de estrategias y herramientas avanzadas para el análisis y trading de ETFs utilizando el lenguaje de programación R. Incluye scripts para optimización, análisis multivariable y exportación de resultados profesionales.

## Estructura del Proyecto 
- `Estrategia-ETF-Pro.R`: Estrategia profesional robusta con machine learning (caret, ranger).
- `EstrategiaETF-Multiples.R`: Análisis de portafolios de múltiples ETFs.
- `EstrategiaETF-Optimizada.R`: Implementación enfocada en alto rendimiento.
- `ExportarResumen.R`: Herramienta para generar reportes en formato Excel (openxlsx).

## Instalación
Para utilizar estos scripts, asegúrese de tener instalado R y las dependencias necesarias.

### Requisitos
- R >= 4.0.0
- Bibliotecas principales:
  - `quantmod`, `PerformanceAnalytics`, `TTR`, `xts`
  - `tidyverse` (dplyr, ggplot2, tidyr, etc.)
  - `caret`, `ranger`, `randomForest`
  - `plotly`, `corrplot`
  - `openxlsx`, `pacman`

## Uso
Se recomienda el uso de RStudio. Muchos scripts utilizan `pacman` para gestionar las dependencias automáticamente en la primera ejecución.
